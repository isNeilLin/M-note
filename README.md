# m-note
![](https://travis-ci.org/isNeilLin/M-note.svg?branch=master)

> 一个基于 electron-vue 的Markdown编辑器

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
