# m-note
![](https://travis-ci.org/isNeilLin/M-note.svg?branch=master)

> 一个基于 electron-vue 的Markdown编辑器

#### 构建步骤

``` bash

# 下载源码
git clone git@github.com:isNeilLin/M-note.git

# 切换到项目跟路径
cd m-note

# 安装依赖
npm install
# or
yarn

# 开发环境
npm run dev

# 生产环境打包
1、npm run pack
2、npm run build


```
